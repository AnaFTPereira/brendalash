# Mobile Application

This is the Expo/React Native mobile application for the monorepo.

## Accessibility

- Screen reader support: All interactive elements have accessible labels, and the app is navigable using screen readers like VoiceOver (iOS) and TalkBack (Android).
- Accessible labels: All form inputs, buttons, and other interactive elements have clear and descriptive labels.
- Visual focus: A clear visual focus indicator is provided for all interactive elements to assist users who navigate with a keyboard or other assistive devices.
